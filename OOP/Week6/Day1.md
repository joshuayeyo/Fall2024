### 비정방형배열
- 각 행의 열의 개수가 다른 배열
- 비정방형 배열의 생성
```
int i[][];
i = new int[4][];

i[0] = new int[1];
i[1] = new int[2];
i[2] = new int[3];
i[3] = new int[4]
```
