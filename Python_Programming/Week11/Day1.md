## 8.4 정보은닉
### 캡슐화 (encapsulation)
- 객체의 속성(데이터)과 메소드(동작)을 외부에서 직접 접근하지 못하도록 숨기고,
- 필요한 인터페이스를 통해서만 접근하도록 제한하는 방식
- 데이터 보호: 예기치 않은 오류나 잘못된 수정 방지
- 접근 제한: 코드의 복잡성 낮추고 관리의 편의성 확보
- 객체간 결합도 감소: 제한된 인터페이스(메소드)를 통해서만 상호 작용하므로 불필요한 결합 방지
### 정보은닉(information hiding)
- 세부사항을 클래스 안에 감추는 것 (클래스 안의 데이터를 외부에서 조작하지 못하도록 함)
- 클래스 내부의 변수를 private 변수로 선언 (외부로부터의 접근 차단)
### 변수 이름 정의 
```
class Student:
	def _init_(self, name=None, age=0):
		self._name = name
		self._age = age
obj = Student()
print(obj._age)
```
### 인스턴스변수 접근(메소드)
- 접근자: 인스턴스 변수값을 반환 
	`e.g. getAge()`
- 설정자: 인스턴스 변수값을 설정 
	`e.g. setAge()`

## 8.5 객체참조
- 변수, 객체, 메모리 주소
- 객체 참조값(object reference): 객체의 주소
### 참조 공유
- 객체의 참조값을 저장하고 있는 변수를 다른 변수로 복사
	```
	# 참조값 복사
	t = Television(11, 10, True)
	s = t
	```
- s를 통하여 객체 수정
```
	# t가 참조하는 객체값 변경
	t = Television(11, 10, True)
	s = t
	s.channel = 9 # t를 변경한 것은 아니지만, t도 변경된다. t가 참조하고 있는 객체값을 변경하는 것이기 때문이다.
```
### is, is not
- 2개의 변수가 동일한 객체를 참고하고 있는지를 검사
```
	if s is t:
	print("2개의 변수는 동일한 객체를 참조하고 있습니다.")
	is s is not t:
	print("2개의 변수는 서로 다른 객체를 참조하고 있습니다.")
```
### None 참조값
- None: 변수가 현재 아무것도 참조하고 있지 않을 때
	- 아무것도 참조하고 있지 않다는 것을 나타내는 특별한 값
```
	myTv = None

	if myTV is None:
		print("현재 TV가 없습니다.)
```

## 8.6 클래스 변수
### 클래스 멤버
- 모든 객체를 통틀어서 하나만 생성
- 모든 객체가 공유하는 변수
### 클래스 안의 변수
- 인스턴스 변수: 반드시 객체를 통해서 사용
- 클래스 변수: 모든 객체를 대상으로 공통으로 사용되는 변수
	- 클래스 내부, 메소드 외부에 변수 생성(인스턴스와 무관)

## 8.7 특수 메소드
### 특수 메소드
- 연산자 `+, -, * , /`
- 객체에 대하여 적용하면 자동으로 호출
```
class Circle:
	...
	def __eg__(self, other):
		return self.radius == other.radius
c1 = Circle(10)
c2 = Circle(10)

if c1 == c2:
	print("원의 반지름은 동일합니다.")
else:
	print("원의 반지름은 상이합니다.")
```
